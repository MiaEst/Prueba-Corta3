Tema 1

HTML

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Utiliza etiquetas para definir elementos como encabezados, párrafos, enlaces, imágenes y más. Los navegadores web interpretan el código HTML para mostrar páginas web de manera visual.

Java Script

¿Qué es JavaScript? JavaScript es un lenguaje de programación de alto nivel y multiparadigma que se utiliza principalmente para crear interactividad en sitios web. Es uno de los tres pilares fundamentales del desarrollo web, junto con HTML y CSS. Originalmente diseñado para el desarrollo del lado del cliente, hoy en día también se utiliza en el servidor gracias a plataformas como Node.js.

CSS

¿Qué es CSS? CSS (Cascading Style Sheets) es el lenguaje que define el estilo y la presentación de las páginas web. Permite controlar colores, fuentes, diseños y adaptabilidad en diferentes dispositivos. Usos Principales: Dar formato visual al contenido HTML. Crear diseños responsivos y atractivos. Personalizar elementos como botones y menús.

Frameworks

¿Qué es un Framework? Un framework es una estructura de software que proporciona herramientas, bibliotecas y reglas para facilitar el desarrollo de aplicaciones. Ayuda a organizar el código y agiliza tareas comunes, como la gestión de rutas o la conexión a bases de datos.
